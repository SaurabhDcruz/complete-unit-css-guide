📐 Complete CSS Units Guide

1. Absolute Units (Fixed - Physical Measurements)

px - Pixels
100px wide
200px wide
width: 200px;
1px = 1/96th inch (screen pixel)
pt - Points
72pt wide
20pt text
font-size: 20pt;
1pt = 1/72 inch (Print media)
pc - Picas
6pc wide
2pc text
width: 6pc;
1pc = 12pt = 1/6 inch
in - Inches
1in wide
2in wide
width: 1in;
Physical inch (depends on screen DPI)
cm - Centimeters
2cm wide
4cm wide
width: 2cm;
Physical centimeter
mm - Millimeters
20mm wide
40mm wide
width: 20mm;
Physical millimeter (10mm = 1cm)

2. Relative Units (Font Based)

rem - Root EM
1rem (16px default)
1.5rem (24px)
width: 10rem
font-size: 1.5rem;
Relative to root (<html>) font-size
em - EM (Parent relative)
Parent: 20px
1.5em = 30px
2em = 40px
font-size: 1.5em;
Relative to parent element's font-size
ex - x-height
x
2ex text
height: 3ex;
Relative to 'x' character height
ch - Character unit
20ch wide text here
30ch wide text example
width: 20ch;
Width of '0' character

3. Viewport Units (Screen Relative)

vw - Viewport Width
25% of viewport width
5vw text
width: 25vw;
1vw = 1% of viewport width
vh - Viewport Height
15vh height
5vh text
height: 15vh;
1vh = 1% of viewport height
vmin - Viewport Minimum
20vmin
4vmin text
width: 20vmin;
1vmin = 1% of smaller viewport dimension
vmax - Viewport Maximum
20vmax
4vmax text
width: 20vmax;
1vmax = 1% of larger viewport dimension

4. Percentage Units (%)

Parent: 400px
50% = 200px
75% = 300px
100% = 400px
width: 50%; /* Relative to parent */
Percentage = Relative to parent element's property

5. Fraction Units (CSS Grid Only)

1fr
2fr
1fr
grid-template-columns: 1fr 2fr 1fr;
fr = Fractional unit (divides available space proportionally)

6. Modern CSS Units (New Features)

lvh - Large Viewport Height
10lvh
height: 10lvh;
Largest possible viewport (includes mobile address bar)
svh - Small Viewport Height
10svh
height: 10svh;
Smallest possible viewport (dynamic)
dvh - Dynamic Viewport Height
10dvh
height: 10dvh;
Changes dynamically as address bar appears/disappears
cqw - Container Query Width
50% of container width
width: 50cqw;
Relative to container width (new container queries)
